Web Element Locator

AI Destekli Web Element Seçici ve Locator Oluşturucu Chrome Eklentisi

v1.0.0 Chrome Extension AI Powered
MIT License Open Source

Proje Hakkında

Web Element Locator, web geliştiricileri ve test otomasyonu uzmanları için tasarlanmış gelişmiş bir Chrome eklentisidir. Herhangi bir web sayfasındaki elementleri kolayca seçmenizi ve bu elementler için XPath, CSS Selector gibi locator'ları otomatik olarak oluşturmanızı sağlar.

MIT Lisanslı Açık Kaynak Proje - Bu proje MIT lisansı altında yayınlanmış olup, ticari ve kişisel kullanım için tamamen ücretsizdir.

Öne Çıkan Özellikler

Akıllı Element Seçimi

Sayfadaki herhangi bir elementi kolayca seçin. Element türü otomatik algılanır ve vurgulanır.

Çoklu Locator Üretimi

3 seviyede (Simple, Medium, Advanced) XPath ve CSS Selector otomatik üretimi.

AI Entegrasyonu

ChatGPT, Gemini, DeepSeek ve Claude desteği ile akıllı locator önerileri.

Element Geçmişi

Seçtiğiniz elementlerin geçmişini tutun, ileri-geri gezinin ve temizleyin.

Gelişmiş Güvenlik

Developer tools koruması, sağ tık engeli ve güvenlik ayarları.

Performans İzleme

Rate limiting, circuit breaker ve performans metrikleri ile optimize edilmiş çalışma.

Kod Örnekleri

Örnek Locator Çıktıları

XPath Örnekleri

// Simple
//*[@id="login-button"]

// Medium  
//button[contains(@class, "btn-primary")]

// Advanced
//form[@class="login-form"]//button[
  contains(text(), "Login") and 
  @type="submit"
]

CSS Selector Örnekleri

/* Simple */
#login-button

/* Medium */
button.btn-primary[type="submit"]

/* Advanced */
form.login-form button:nth-child(2)[
  data-testid*="login"
]

Test Otomasyonu Entegrasyonu

Selenium (Python)

from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()

# Eklentiden kopyalanan XPath
login_btn = driver.find_element(
    By.XPATH, 
    "//button[@data-testid='login-submit']"
)
login_btn.click()

Playwright (JavaScript)

const { chromium } = require('playwright');

const browser = await chromium.launch();
const page = await browser.newPage();

// Eklentiden kopyalanan CSS selector
await page.click('button.login-btn[type="submit"]');

await browser.close();

Sık Karşılaşılan Sorunlar

Sidebar açılmıyor

Çözüm:

  • Eklentinin etkin olduğundan emin olun
  • Sayfayı yenileyin (F5)
  • Chrome'u yeniden başlatın

AI önerileri çalışmıyor

Çözüm:

  • API token'ının doğru olduğundan emin olun
  • Rate limit kontrolü yapın
  • İnternet bağlantınızı kontrol edin

Element seçilemiyor

Çözüm:

  • Locator mode'un aktif olduğundan emin olun
  • JavaScript hatalarını konsol'dan kontrol edin
  • Sayfanın tamamen yüklendiğinden emin olun
