Element Locator Chrome Extension Privacy Policy

Introduction

Element Locator ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how information is collected, used, and disclosed when you use our Element Locator Chrome Extension ("Extension").

Information We Collect

Our Extension collects the following information:

  1. Local Storage Data: The Extension stores your settings locally on your device using Chrome's storage API. This information never leaves your device.
  2. Temporary Page Content Access: The Extension temporarily accesses and analyzes the content of web pages you visit when you actively use the Extension to select elements. This data is processed locally and is never sent to our servers.
  3. API Keys: If you choose to use AI suggestion features, you must provide your own API keys for services like Google Gemini, OpenAI, Anthropic Claude, or DeepSeek. These keys are stored locally on your device and are only used to make API calls directly from your browser to the respective AI service providers.

How We Use Information

The information collected is used solely to:

  • Provide element locator functionality
  • Generate AI-powered selector suggestions
  • Provide element locator functionality
  • Save your preferences and settings

Data Sharing and Disclosure

We do not:

  • Collect or store any of your browsing data on our servers
  • Share or sell any data with third parties
  • Transfer any personal information outside your device

The only data that leaves your device are the API requests made directly to the AI service providers you configure (if you choose to use those features). These requests include minimal context about the HTML elements you select.

Data Security

All locally stored data is secured according to Chrome's extension security model. API keys you provide are stored using Chrome's secure storage mechanisms.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy in our GitHub repository and updating the Extension accordingly.
